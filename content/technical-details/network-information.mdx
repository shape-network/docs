## RPC Endpoints

### Mainnet

- HTTPS: `https://rpc.shape.network`
- WebSocket: `wss://ws.shape.network`

### Testnet

- HTTPS: `https://rpc.testnet.shape.network`
- WebSocket: `wss://ws.testnet.shape.network`

## Adding Shape to Your Wallet

### MetaMask

1. Open MetaMask and click on the network selector at the top
2. Click "Add Network"
3. Click "Add a network manually"
4. Fill in the following details:

**Shape Mainnet:**

- Network Name: `Shape Mainnet`
- RPC URL: `https://rpc.shape.network`
- Chain ID: `2231`
- Currency Symbol: `ETH`
- Block Explorer URL: `https://explorer.shape.network`

**Shape Testnet:**

- Network Name: `Shape Testnet`
- RPC URL: `https://rpc.testnet.shape.network`
- Chain ID: `2232`
- Currency Symbol: `ETH`
- Block Explorer URL: `https://testnet.explorer.shape.network`

### Wallet Connect

Shape is compatible with Wallet Connect v2. Applications can connect to Shape using Wallet Connect by selecting the appropriate chain ID.

## Network Status

You can check the current status of the Shape network, including block time, gas prices, and any ongoing maintenance at our [status page](https://status.shape.network).

## Bridge Access

To transfer assets between Ethereum and Shape, use our official bridge:

- [bridge.shape.network](https://bridge.shape.network)

## API Rate Limits

Public RPC endpoints have the following rate limits:

- 10 requests per second
- 200 requests per minute
- 10,000 requests per day

For higher limits, please consider using one of our [Node Providers](/node-providers).
