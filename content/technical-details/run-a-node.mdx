# Run a Node

Learn how to run your own Shape Mainnet and Shape Sepolia nodes.

Follow [these instructions](https://github.com/ethereum-optimism/optimism/blob/develop/specs/rollup-node.md) to learn how to run your own Shape node.

The process is the same as running a standard op-node, but using a different `rollup.json` and `genesis.json`.
