# Run a Node

Learn how to run your own Shape Mainnet and Shape Sepolia nodes.

Follow [these instructions](https://github.com/ethereum-optimism/specs/blob/main/specs/protocol/rollup-node.md) to learn how to run your own Shape node.

The process is the same as running a standard op-node, but using a different `rollup.json` and `genesis.json`, except you will need to add the following commands
