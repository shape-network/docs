# Node Providers

| Provider | Endpoint | Notes |
| --- | --- | --- |
| Public RPC | `https://mainnet.shape.network` | Rate-limited, not for production |
| Alchemy | `https://shape-mainnet.g.alchemy.com/v2/YOUR_KEY` | Higher limits, WebSocket, enhanced APIs |

Sepolia: `https://sepolia.shape.network` (public) or `https://shape-sepolia.g.alchemy.com/v2/YOUR_KEY` (Alchemy).

WebSocket: `wss://shape-mainnet.g.alchemy.com/v2/YOUR_KEY`

The public RPC is rate-limited and not suitable for production. Use [Alchemy](https://www.alchemy.com/rpc/shape) (free tier available).
